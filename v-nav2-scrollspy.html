Scrollspy & Affix Example

Fixed vertical sidenav on scroll

Scroll this page to see how the navigation bar behaves with the data-is-sticky="true" attribute set on the unordered list to be affixed, a data-sticky-fix-at-top (no value necessary) attribute set on the element whose top is the indicator as to when to affix the navigation (when it scrolls out of view), and a data-sticky-bottom-boundary attribute set on the element whose top will serve as the bottom boundary for sizing the navigation during window/viewport resizing.

The left menu sticks to the page after you have scrolled to where the element with a data-sticky-fix-at-top attribute set on it has its top at the top viewport edge, and the links in the menu are automatically updated to show the scroll content in view based on scroll position.
